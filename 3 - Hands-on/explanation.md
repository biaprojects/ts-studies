Exercício — Sistema de Cadastro Simples (Terminal)

Objetivo  
Construir um pequeno sistema no terminal que gerencia uma lista de usuários.

Cada pessoa terá:
- nome
- idade

Requisitos
- Você deve armazenar os usuários em um array.
- Seu programa deve permitir:
    1. Cadastrar usuário
    2. Pedir nome
    3. Pedir idade
    4. Adicionar no array
    5. Mostrar todos os usuários cadastrados
    6. Filtrar maiores de idade (usando `.filter()`)
    7. Buscar usuário por nome (usando `find()`)
    
O programa não pode rodar só uma vez.
Ele deve repetir algo assim:
1 - Cadastrar  
2 - Listar  
3 - Filtrar maiores de idade  
4 - Buscar por nome  
5 - Sair
